Markdown 笔记

发表于 2024-08-13

一、标题

表示方式：#(几级标题对应几个#)

二、内容

1.段落

直接编写即可
tip：换行：double空格+回车or直接空一行

2.字体

斜体：用*or_围住即可
粗体：用double用*or_围住即可

3.分割线

3or6个*or-

4.删除线

用double~~围住即可

5.下划线

使用HTML标签<u></u>围住

6.脚注

[^目标文本]:脚注内容

列表

  1. 无序列表
    • *or-or+跟小标题
  2. 有序列表（嵌套）
    • 1.+ *

三、区块

表示方式：>加内容
比如这是区块

这是二级区块

四、代码块

  1. 小段代码
    • 用’’包住
  2. 代码块
    • Tab
  3. 显示语言
    • ‘’’+语言+回车来包住

链接

  • 链接名称
  • <链接地址>
  • 赋值法：

    [变量名][变量]
    [变量]:链接

图片

  • 变量法同上
  • 使用HTML的标签

表格

|  表头   | 表头  |
|  ----  | ----  |
| 单元格  | 单元格 |
| 单元格  | 单元格 |
  • 对齐

    -：靠右
    :- 靠左
    :-: 居中
    | 左对齐 | 右对齐 | 居中对齐 |
    | :—–| —-: | :—-: |(只能写在这里)
    | 单元格 | 单元格 | 单元格 |
    | 单元格 | 单元格 | 单元格 |
